# Example Prompts

Things you can ask Simon:

## AI Agents
- What AI Agent did I recently update?
- Please add an incident open tool to my DUG agent.
- Show me all the tools configured on my HR agent.
- Which AI Agents use the Incident table?
- Clone the tools from my DUG agent to my HR agent.
- Which of my AI Agents have no topics assigned?

## Virtual Agent
- What topics are assigned to my IT Support Virtual Agent?
- Find any Virtual Agent conversations that errored out today.

## Flow Designer
- What Flow Designer flows were modified in the last 7 days?

## Business Rules & Script Includes
- List all active business rules on the Incident table.
- What script includes have I changed recently?

## Update Sets
- Show me all Update Sets that are in progress.
