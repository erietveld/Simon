# AI Agent — External Agents (A2A Protocol) — Hints

## Key Tables

| Table | Label | Purpose |
|---|---|---|
| `sn_aia_external_agent_configuration` | External Agent Configuration | Config for connecting to an external agent |
| `sn_aia_external_agent_provider` | External Agent Provider | Provider registry for external agents |
| `sn_aia_external_agent_protocol` | External Agent Protocol | Protocol definitions (e.g. A2A) |
| `sn_aia_external_agent_skill` | External Agent Skill | Skills exposed by external agents |
| `sn_aia_external_agent_discovery` | External Agent Discovery | Discovery config for external agents |
| `sn_aia_external_agent_card` | External Agent Card | Agent card metadata |
| `sn_aia_external_agent_callback_registry` | External Agent Callback Registry | Registered callbacks for external agents |
| `sn_aia_ea_push_notification_url` | External Agent Push Notification | Push notification endpoint URLs |
| `sn_aia_external_agent_session` | External Agent Session | Active session records (runtime) |
| `sn_aia_external_agent_execution` | External Agent Execution | Execution records for external agent calls (runtime) |
| `sn_aia_external_agent_exec_history` | External Agent Exec History | History of external agent executions (runtime) |

> **Ignore:** `*_lab` suffix variants (e.g. `sn_aia_external_agent_protocol_lab`) are UI label tables, not data tables.
