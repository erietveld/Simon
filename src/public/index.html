ServiceNow OAuth Setup
  1. Open the Machine Identity Console in ServiceNow (enter instance URL above first)
  2. Click New integration → select OAuth – Authorization code grant
  3. Set Redirect URL to http://localhost:3001/auth/callback copied
  4. Copy the generated Client ID and Client Secret into the fields below
  5. Set the Auth scope to useraccount and save the record
  6. After adding this instance, click the Login button
